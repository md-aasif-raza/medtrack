C++ OOP Concepts

Tech Behind the Project

Real OOP concepts mapped to real project features

🏗️

Classes & Objects

Medicine and HealthRecord are classes. Every entry you add is an object — a real-world instance with its own data and methods.

class Medicine {
  string name, dosage, time;
  int duration;
  void display();
};
// Object creation:
Medicine m1; m1.name = "Paracetamol";
🧬

Inheritance

HealthRecord extends Medicine — inherits base properties while adding vitals-specific attributes like metric, unit, and status.

class HealthRecord : public Medicine {
  string metric;
  string value, unit;
  string date, status;
};
🔒

Encapsulation

Private data members protect health info. Only class methods can read or modify them — data safety is built into the design.

private:
  string name;
  int dosage;
public:
  string getName() {
    return name;
  }
⚙️

Operator Overloading

The == operator is overloaded to compare two medicines — automatically prevents duplicate entries from being added.

bool operator==(Medicine &m) {
  return name == m.name;
}
// Usage:
if(m1 == m2) cout << "Duplicate!";
